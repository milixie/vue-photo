# vue-Demos

> 使用vue制作的一些demo

## 克隆启动项目

``` 
git clone git@github.com:milixie/vue-photo.git

cd vue-photo

npm install

# serve with hot reload at localhost:8080
npm run dev
```
可以download项目查看具体的效果

启动项目后

```
// 查看照片墙项目效果
http://localhost:8080/#/photo

// 查看大转盘抽奖系统效果
http://localhost:8080/#/turntable

// 查看方格抽奖系统效果
http://localhost:8080/#/lottery
```
